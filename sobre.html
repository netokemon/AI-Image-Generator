Sobre o Projeto

O projeto NetoAI é uma aplicação web simples desenvolvida para demonstrar o uso de geração de imagens via inteligência artificial. Através de uma interface amigável, o usuário pode digitar um prompt descritivo e gerar uma imagem realista com auxílio de uma API de IA.


Este projeto tem fins educacionais e foi construído utilizando tecnologias como HTML, CSS e JavaScript. O objetivo principal é explorar o uso seguro de chaves de API, o consumo de APIs externas e a organização de um projeto com separação entre frontend e backend.
